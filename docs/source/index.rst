==================================

|current_plugin| is a plugin for QGIS to work with GeoGig repositories using the
GeoGig Web API.  GeoGig enables managing, controlling, and understanding change 
for GeoSpatial datasets.  It also provides provides high-level workflows for managing change processes as well as lower level geospatial versioning (drawing inspiration from *Git*). 

GeoGig-Core is a `Locationtech <https://www.locationtech.org/>`_  project, for which Boundless
(http://boundlessgeo.com/) contributes actively. For more information about
GeoGig see http://geogig.org/

The GeoGig-WebAPI and GeoGig-QGIS-plugin are `Boundless Geo <https://boundlessgeo.com/>`_ projects.
