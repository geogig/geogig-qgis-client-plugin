Overview
--------

GeoGig is a tool for managing change in geospatial datasets with sophisticated workflows and version management.  The easiest way to use GeoGig is through the QGIS GeoGig plugin.  For more information see the `Introduction to GeoGig <https://vimeo.com/295261790>`_ and `GeoGig Quick Start <https://vimeo.com/298415397>`_  videos.

This Quick Start is designed to get you quickly using GeoGig and the GeoGig-QGIS3 plugin.

Please view the video, above, which walks through this Quick Start.
